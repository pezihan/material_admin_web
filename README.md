# material_admin_web

**介绍**

本项目为基于 material_server服务api的后台管理，用来管理网页端用户及作品的管理，已打包成静态资源放到了material_server下，这个是 vue 源码

**管理端默认账户： admin  密码：123456**

material_server 服务端项目地址： https://github.com/pezihan/material_server

## 安装依赖
```
npm install
```

### 运行
```
npm run serve
```

### 打包静态资源
```
npm run build
```

### 代码纠错
```
npm run lint
```
